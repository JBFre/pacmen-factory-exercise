# Pacman Game
This is a simple Pacman game implemented in JavaScript, HTML, and CSS. The game allows you to add multiple Pacman characters to the screen, and they move around, bouncing off the window boundaries.

## Getting Started:
1. Clone the repository or download the source code files.
2. Open the index.html file in a web browser.
3. Click the "Add PacMan" button to add Pacman characters to the screen.
4. Click the "Start Game" button to begin the game loop and watch the Pacman characters move around.

## Game Instructions:
- Click the "Add PacMan" button to add more Pacman characters to the screen. They will appear at random positions with random velocities.
- Click the "Start Game" button to start the game loop. The Pacman characters will continuously move around the screen, bouncing off the window boundaries.
- The Pacman characters change their appearance between open and closed mouths as they move.
- If a Pacman character reaches the edge of the window, it will bounce back in the opposite direction.
